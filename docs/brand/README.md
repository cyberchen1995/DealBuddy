# DealBuddy 购物搭子品牌说明

DealBuddy 的中文名是「购物搭子」。

这个名字强调的是陪伴式决策，而不是促销、抢券或自动下单。它像一个坐在旁边一起看商品页的人，帮你把页面证据收好、把参数讲清、把价格条件标出来，最后给出有边界的建议。

## 视觉方向

Logo 避免购物车、优惠券、价格标签等促销符号，改用「商品卡片 + 对话气泡 + 决策路径 + 勾选」表达产品气质：

- 商品卡片：页面事实和手动采集
- 对话气泡：搭子式陪伴和需求澄清
- 蓝色路径：从信息噪音走向可解释选择
- 琥珀勾选：最后建议和可信结论

## 文案语气

DealBuddy 购物搭子的文案应该清楚、克制、可信：

- 说“整理商品事实”，不要说“全网最低价”。
- 说“估算应付”，不要说“结算价”。
- 说“本地优先”，不要承诺绝对隐私。
- 说“帮你做选择”，不要暗示替用户下单。
