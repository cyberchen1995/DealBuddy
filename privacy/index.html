隐私政策

生效日期：2026-06-21

DealBuddy 购物搭子是一款 Chrome 扩展，用于在用户访问淘宝、天猫或京东商品详情页时，帮助用户整理当前商品页面中可见的商品信息，并通过浏览器本地 OCR 识别商品详情图片中的文字，辅助用户进行商品比较和购物决策。

01我们处理哪些数据

当用户主动点击「整理当前商品信息」，或开启自动整理功能后，本扩展可能处理当前商品详情页中的以下内容：

  • 商品平台、商品页面 URL、商品标题
  • 页面可见价格、店铺名称
  • 商品主图、相册图片 URL、详情图片 URL
  • SKU、已选择规格、商品参数、页面详情文字
  • 本地 OCR 从商品详情图片中识别出的文字
  • 采集时间、OCR 状态、处理警告和置信度

本扩展不会主动收集用户姓名、邮箱、电话号码、身份证件、支付信息、账户密码、健康信息、精确地理位置或个人通信内容。

02数据用途

本扩展处理上述数据仅用于：

  • 整理当前商品详情页的结构化信息
  • 在浏览器本地识别商品详情图片中的文字
  • 将整理结果发送到用户配置的本机 DealBuddy intake 服务
  • 在发送失败时，允许用户复制整理后的 JSON
  • 保存用户设置并提升重复 OCR 的处理效率

本扩展不会将数据用于广告投放、用户画像、跨站跟踪、信用评估、贷款评估或与商品整理无关的用途。

03数据存储

本扩展会使用 Chrome 扩展本地存储保存用户设置，包括自动整理开关、详情图片加载等待时间和本机 intake 服务地址。

本扩展还可能使用浏览器 IndexedDB 缓存 OCR 结果。缓存内容包括图片 URL、该图片识别出的文字和更新时间。该缓存仅用于减少重复 OCR，提高本地处理效率。

04数据传输

默认情况下，整理后的商品信息会发送到用户本机运行的 DealBuddy intake 服务，默认地址为：

http://127.0.0.1:8765/api/current/offers

用户可以在扩展 popup 中修改该地址。请仅填写自己信任的服务地址。

为完成本地 OCR，本扩展会下载当前商品页引用的商品图片。图片请求仅限受支持电商平台及其常用图片 CDN 域名，包括 alicdn.com、tbcdn.cn、jd.com、360buyimg.com 和 jdimg.com 等。图片下载用于在浏览器本地识别文字，不会携带网站 cookie。

05数据共享

本扩展不会出售用户数据。

本扩展不会将用户数据分享给广告网络、数据经纪商或无关第三方。

除用户主动配置的本机服务地址外，本扩展不会默认将整理后的商品信息发送到第三方云端服务。

06用户控制

用户可以通过以下方式控制数据处理：

  • 在扩展 popup 中关闭自动整理功能
  • 不点击「整理当前商品信息」
  • 修改本机 intake 服务地址
  • 在 Chrome 扩展管理页面移除本扩展
  • 通过浏览器设置清除扩展本地存储和 IndexedDB 缓存

07权限说明

  • storage 权限用于保存扩展设置。
  • 淘宝、天猫、京东商品页权限用于读取当前商品详情页中可见的商品内容。
  • 图片 CDN 域名权限用于下载商品详情图片并在浏览器本地执行 OCR。
  • 127.0.0.1 和 localhost 权限用于将用户主动整理的商品信息发送到用户本机运行的 DealBuddy intake 服务。

08儿童隐私

本扩展不面向儿童设计，也不会有意收集儿童个人信息。

09政策更新

如果本扩展的数据处理方式发生变化，我们会更新本隐私政策。

10联系方式

如对本隐私政策或数据处理方式有疑问，请通过 Chrome Web Store 开发者页面提供的联系方式与我们联系。
